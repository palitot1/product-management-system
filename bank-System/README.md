## Sistema Bancário em JavaScript (V1)

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação utilizando JavaScript puro, sem o uso de bibliotecas ou frameworks.

A aplicação simula operações básicas de um sistema bancário, permitindo cadastrar clientes, realizar depósitos, saques, transferências e consultar saldos.

## Funcionalidades
- Cadastro de clientes
- Depósito em conta
- Saque com validação de saldo
- Transferência entre clientes
- Consulta de saldo
- Listagem de clientes cadastrados

## Tecnologias

JavaScript (ES6)

## Como executar

Clone o repositório:

git clone https://github.com/palitot1/javascript-practice.git

Acesse a pasta do projeto.
Execute o arquivo com o Node.js:
node index.js
