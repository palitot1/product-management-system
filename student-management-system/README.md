#  Student Management System

Este projeto foi desenvolvido durante meus estudos de JavaScript para praticar lógica de programação. A ideia foi criar um sistema simples capaz de cadastrar alunos, calcular suas médias e informar se foram aprovados ou reprovados.

## Funcionalidades

- Cadastrar alunos
- Calcular a média das notas
- Listar alunos aprovados
- Buscar um aluno pelo nome
- Exibir a situação do aluno (Aprovado ou Reprovado)

##  Tecnologias utilizadas

- JavaScript


## ▶️ Como executar

### Pré-requisitos

- Ter o Node.js instalado.

### Passos

1. Clone o repositório:

```bash
git clone https://github.com/palitot1/student-management-system.git
```

2. Entre na pasta do projeto:

```bash
cd student-management-system
```

3. Execute o arquivo:

```bash
node index.js
```
